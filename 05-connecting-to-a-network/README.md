# 05 — Connecting to a Network

Primary reference: *Practical Linux System Administration*, Chapter 5.

### Lab

`lab-05-configure-and-troubleshoot-linux-networking.md`

### Focus

Configure and troubleshoot network connectivity, static and dynamic addressing, SSH access, network daemon exposure, and basic host network security.

### Augmentation

Use TCP/IP, DNS, routing, NetworkManager, Netplan, packet inspection, service exposure review, and operational network documentation.
