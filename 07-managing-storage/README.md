# 07 — Managing Storage

Primary reference: *Practical Linux System Administration*, Chapter 7.

### Lab

`lab-07-manage-linux-storage-filesystems-and-lvm.md`

### Focus

Inspect disks, create filesystems, mount storage, configure persistent mounts, manage swap and tmpfs, implement LVM, and handle decommissioning decisions.

### Augmentation

Use filesystem theory, block device concepts, fstab failure recovery, data lifecycle, storage monitoring, and operational risk controls.
