# 07 — Security and Hardening

This topic covers Linux host security and baseline hardening.

### Focus Areas

* SSH hardening
* host firewalls
* SELinux on RHEL
* AppArmor on Ubuntu
* service exposure review
* local account review
* least privilege
* baseline audit checks

### Example Labs

* `lab-01-harden-ssh-and-firewall-access.md`
* `lab-02-selinux-vs-apparmor-service-access.md`

### Operational Standard

A lab in this folder should prove that security controls are understood, documented, tested, and not disabled permanently just to make a service work.
