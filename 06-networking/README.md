# 06 — Networking

This topic covers Linux host networking and troubleshooting.

### Focus Areas

* IP addressing
* routing
* DNS resolution
* hostnames
* Netplan on Ubuntu
* NetworkManager and `nmcli` on RHEL
* listening ports
* packet capture
* network failure diagnosis

### Example Labs

* `lab-01-troubleshoot-dns-resolution.md`
* `lab-02-configure-static-networking.md`

### Operational Standard

A lab in this folder should prove the difference between local configuration, name resolution, routing, firewalling, and application-level connectivity.
