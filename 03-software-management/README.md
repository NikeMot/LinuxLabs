# 03 — Software Management

This topic covers package installation, repositories, updates, and package troubleshooting.

### Focus Areas

* `apt` on Ubuntu
* `dnf` on RHEL
* repository configuration
* GPG trust
* package ownership queries
* updates and rollbacks
* security patching decisions

### Example Labs

* `lab-01-configure-package-repositories.md`
* `lab-02-patch-and-verify-linux-servers.md`

### Operational Standard

A lab in this folder should prove that software changes are controlled, explainable, reversible where possible, and verified after reboot.
