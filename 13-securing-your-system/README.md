# 13 — Securing Your System

Primary reference: *Practical Linux System Administration*, Chapter 13.

### Lab

`lab-13-harden-and-audit-linux-system-security.md`

### Focus

Protect privileged access, reduce attack surface, secure user accounts, apply hardening measures, install security tools, and document incident response decisions.

### Augmentation

Use policy thinking, STIG-style controls, SELinux/AppArmor comparison, host firewalling, security incident handling, and responsible AI safety review.
