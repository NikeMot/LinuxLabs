# 02 — Working with Permissions and Privileged Accounts

Primary reference: *Practical Linux System Administration*, Chapter 2.

### Lab

`lab-02-manage-permissions-and-privileged-access.md`

### Focus

Work as a regular user, control privileged access, use `sudo` safely, read and modify file permissions, apply symbolic and numeric modes, and explain `umask`.

### Augmentation

Use POSIX permission theory, shell inspection commands, least-privilege thinking, and production access-control standards.
