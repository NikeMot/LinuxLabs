# 01 — Getting Started with Linux

Primary reference: *Practical Linux System Administration*, Chapter 1.

### Lab

`lab-01-bootstrap-and-baseline-linux-server.md`

### Focus

Install or validate Linux, inspect the system, practise CLI basics, confirm SSH access, reboot safely, and document the baseline.

### Augmentation

Use command-line depth from *The Linux Command Line*, modern context from *Learning Modern Linux*, OS boot/process concepts from operating systems references, and execution discipline from *The Practice of System and Network Administration*.
