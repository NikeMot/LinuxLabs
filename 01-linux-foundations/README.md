# 01 — Linux Foundations

This topic covers the foundation needed before touching production Linux systems.

### Focus Areas

* RHEL and Ubuntu installation assumptions
* shell navigation
* filesystem layout
* command-line investigation
* documentation habits
* baseline system inventory
* SSH access basics
* post-reboot verification

### Example Labs

* `lab-01-build-and-baseline-linux-servers.md`
* `lab-02-filesystem-and-command-line-investigation.md`

### Operational Standard

A lab in this folder should prove that the system can be built, accessed, inspected, documented, and understood without relying on a GUI or unexplained commands.
