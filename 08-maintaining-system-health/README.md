# 08 — Maintaining System Health

Primary reference: *Practical Linux System Administration*, Chapter 8.

### Lab

`lab-08-maintain-linux-system-health.md`

### Focus

Clean temporary and shared areas, control home directory sprawl, apply patching processes, review users and groups, maintain naming conventions, and create health reports.

### Augmentation

Use operational hygiene, quota management, patch windows, account retention, health reporting, and production maintenance standards.
